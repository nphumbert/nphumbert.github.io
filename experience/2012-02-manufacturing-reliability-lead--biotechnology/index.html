Manufacturing & Reliability Lead — Biotechnology

Description

Resolved a long-standing bottleneck in a critical Sanofi Pasteur vaccine programme that prevented market release due to obsolete verification equipment.

By optimising key measurement parameters, this work enabled safe replacement of the legacy system within weeks and ensured uninterrupted release under strict regulatory constraints.

Project Details

  • Client Type : Global company
  • Industry : Pharmacology
  • Location : France
  • Started on : February 2012
  • Duration : 7 months
